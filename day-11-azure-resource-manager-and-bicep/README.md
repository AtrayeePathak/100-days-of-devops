# Day 11 — Azure Resource Manager, Templates, and Bicep

**Date:** 2026-07-15

## What I set out to learn

Understand how Azure deployments can be made repeatable and version-controlled through Resource Manager, templates, and Bicep.

## What I did

- Reviewed Azure Resource Manager as the deployment engine behind Azure resources
- Compared ARM templates, Bicep, Azure CLI, and Terraform as options for infrastructure automation
- Studied why declarative templates help reduce manual setup and drift
- Looked at the value of managing infrastructure as code rather than clicking through the portal

## What I learned / key takeaways

- ARM and Bicep make Azure deployments more repeatable and easier to review in source control
- Infrastructure-as-code is a core DevOps skill because it improves consistency and auditability
- A template-based workflow is much better for team environments than manual provisioning
- Learning these tools now makes later cloud automation topics much easier to grasp
