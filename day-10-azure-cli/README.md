# Day 10 — Azure CLI

**Date:** 2026-07-15

## What I set out to learn

Learn how to use Azure CLI to create and manage resources from the terminal instead of only using the portal.

## What I did

- Reviewed the basics of installing and authenticating to Azure CLI
- Looked at common commands for resource groups, virtual machines, and storage accounts
- Compared CLI-based workflows with portal-based workflows
- Practiced thinking about how automation becomes easier when infrastructure actions are scripted

## What I learned / key takeaways

- Azure CLI is one of the fastest ways to work with Azure for DevOps and automation tasks
- Command-line workflows are easier to repeat, document, and integrate into pipelines
- Understanding CLI basics makes it easier to move into IaC tools later
- Portal UI is helpful for learning, but CLI is better for repeatable operations
