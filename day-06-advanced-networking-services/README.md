# Day 06 — Advanced Networking Services

**Date:** 2026-07-11

## What I set out to learn

Understand the higher-level networking services that make Azure architectures resilient and production-ready.

## What I did

- Reviewed Azure Application Gateway and the purpose of a WAF
- Looked at Azure Load Balancer and how it distributes traffic across instances
- Studied Azure DNS for name resolution and public/private zone scenarios
- Reviewed Azure Firewall and the value of centralized security policy
- Looked briefly at VNet peering and VPN Gateway for hybrid connectivity

## What I learned / key takeaways

- Application Gateway is more than simple load balancing; it can also provide layer 7 features and WAF protection
- Load balancers and gateways serve different traffic-routing purposes in Azure
- Azure Firewall is a strong central control point for ingress and egress policies
- Hybrid networking becomes important when on-premises systems need to connect to Azure resources
