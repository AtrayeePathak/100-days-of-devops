# Day 08 — Azure Interview Questions (Compute & Networking)

**Date:** 2026-07-13

## What I set out to learn

Reinforce the core Azure concepts covered so far by reviewing common interview-style questions around compute and networking.

## What I did

- Reviewed cloud computing basics such as scalability, elasticity, and high availability
- Revisited Azure fundamentals including regions, availability zones, and resource organization
- Summarized the difference between IaaS, PaaS, and SaaS in Azure terms
- Practiced explaining Azure networking topics like VNets, subnets, NSGs, and load balancing

## What I learned / key takeaways

- Interview questions are a great way to identify knowledge gaps quickly
- Explaining a concept clearly is often more valuable than memorizing vocabulary alone
- Core Azure topics should be reviewed repeatedly because they appear across many real-world solutions
- Strong fundamentals make later topics like DevOps, AKS, and IaC much easier to understand
